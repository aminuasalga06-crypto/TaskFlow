# Contributing to TaskFlow

Welcome to the TaskFlow project! This guide explains how to set up your environment and submit your contribution.

## 🛠 What You Need

- A GitHub account
- [Git](https://git-scm.com/downloads) installed on your computer
- [VS Code](https://code.visualstudio.com/) (recommended)
- A web browser (Chrome, Firefox, Edge — any works)

## 📋 Step-by-Step Guide

### 1. Fork the Repository
- Go to the main repository on GitHub
- Click the **Fork** button (top right)
- This creates your own copy of the project

### 2. Clone Your Fork
```bash
git clone https://github.com/awwalumar98-max/TaskFlow.git
cd taskflow
```

### 3. Create Your Branch
Use this naming format: `feature/member-[number]-[short-task-name]`

```bash
git checkout -b feature/member-7-add-task
```

### 4. Make Your Changes
- Open the project folder in VS Code
- Find your assigned section (marked with your member number)
- Write your code only inside your section
- Open `index.html` in your browser to test

### 5. Commit Your Changes
```bash
git add .
git commit -m "Member 7: Add new task functionality"
```

### 6. Push to Your Fork
```bash
git push origin feature/member-7-add-task
```

### 7. Open a Pull Request
- Go to your fork on GitHub
- Click **Compare & pull request**
- Add a short description of what you did
- Submit the PR — the leader will review and merge it

## ⚠️ Important Rules

- Only edit your assigned section
- Do not rename or delete any existing functions or IDs
- Test in browser before submitting your PR
- One PR per member

## 💬 Questions?

Reach out to the group leader on your group chat.
